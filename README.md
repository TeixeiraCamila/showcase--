## Tecnologias

- **HTML5** - Semântico, acessibilidade
- **CSS3** - Flexbox, Grid, Custom Properties, Animações
- **JavaScript ES6+** - Vanilla, sem frameworks
- **GSAP** - Animações profissionais (alguns projetos)

## Quick Start

```bash
# Clone
git clone https://github.com/TeixeiraCamila/showcase.git
cd showcase

# Servidor local
python -m http.server 8000

# ou
npx http-server
```

Acesse: `http://localhost:8000`
